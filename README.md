# Comitê de Agrupamentos com Métodos de Otimização

Este repositório contém implementações relacionadas ao problema de **comitê de agrupamentos (cluster ensemble)**, com foco na aplicação de **métodos de otimização**.

## Estrutura do projeto

datasets/
    Conjunto de bases de dados utilizadas nos experimentos.

src/
    Código-fonte do projeto.

    load_datasets.py
        Script responsável por carregar todas as bases de dados utilizadas nos experimentos.

## Bases de dados

As bases utilizadas neste projeto são provenientes das seguintes fontes:

**UCI Machine Learning Repository**
https://archive.ics.uci.edu/ml/index.php

**SIPU Shape Datasets**
http://cs.uef.fi/sipu/datasets/
